# Export

**Status**: Draft

## Description

Export the current filtered document set as CSV or JSON.

## Capabilities

### P1 - Must Have

- Export current filter results as JSON
- Export current filter results as CSV
- Export to file (default: `./monq-export-{collection}-{timestamp}.{ext}`)

### P2 - Should Have

- Copy to clipboard instead of file
- Accessible via Ctrl+P command palette
- Progress indicator for large exports
