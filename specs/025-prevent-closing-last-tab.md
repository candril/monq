# Prevent Closing Last Tab

**Status**: Draft

## Description
Pressing `d` when only one tab is open should show a toast instead of closing it. The last tab cannot be closed.

## Capabilities

### P1 - Must Have
- `d` on the last tab shows a toast: "Cannot close the last tab"
- Tab is not closed

## Out of Scope
- Rearranging or merging tabs
